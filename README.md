# BarberShop Pro

A multi-page barbershop website built with HTML, Bootstrap 5, and vanilla JavaScript.

## Pages

- **index.html** — Home page with hero, about snippet, gallery, pricing preview, and contact form
- **about.html** — Team profiles and salon values
- **pricing.html** — Full service menu with pricing cards and promo info
- **contact.html** — Contact form and location details

## Tech Stack

- [Bootstrap 5.3](https://getbootstrap.com/) — layout, components, and utility classes
- [Bootstrap Icons 1.11](https://icons.getbootstrap.com/) — icon set
- [Google Fonts](https://fonts.google.com/) — Oswald + Open Sans
- Vanilla JS (`main.js`) — form validation and interactive behaviour

## Running Locally

No build step required. Just open any `.html` file directly in your browser, or serve the folder with any static server:

```bash
npx serve .
```
